# libadwaita

This project contains libadwaita (GNOME-specific widgets for GTK4) packaged for build2.

Note that it currently lacks certain automated recipes, such as ones for
compiling SCSS to CSS and addressing glib-mkenums, among others. At some point
we'll want to write specific recipes to manage their generation process, but
for now they're provided manually.

https://gitlab.gnome.org/GNOME/libadwaita
